Bienvenue sur "MaFormationAFPA"

Explorez cet Univers dédié au Développement Web !

Salut à tous les passionnés de code et d'apprentissage ! Je m'appelle Elphège PROISY, je suis actuellemnt étudiant en développement web et web mobile. Je suis ravi de vous accueillir sur mon site, un espace dédié aux codes que j'ai produit durant la formation DWWM à l'AFPA d'Istres.

Un Voyage enrichissant : Naviguer dans le vaste monde du développement web peut être intimidant, surtout lorsque l'on débute avec le HTML, CSS, JavaScript et PHP. C'est pourquoi je partage avec vous ce chemin pédagogique, une exploration détaillée de mes expériences et découvertes.

Préparation Antérieure :

Avant même d'entrer en formation, j'avais déjà commencé à explorer les méandres du développement web. Je découvrais alors avec intérêt le potentiel créatif et la puissance inhérente aux langages tels que HTML, CSS, JavaScript et PHP. Cette préparation initiale a jeté les bases de mon voyage pédagogique, une exploration continue que je partage désormais avec vous.

Algorithmes

Pour commencer la formation, nous avons exploré les concepts fondamentaux des algorithmes. Ces séquences d'instructions permettent de résoudre efficacement des problèmes informatiques. En apprenant à concevoir, analyser et implémenter des algorithmes, j'ai développé une solide base pour aborder les défis de développement web de manière logique et méthodique.

Maquettage :

Mes premiers pas avec Figma.

HTML & CSS :

Les fondements de toute création web. Explorez les exercices HTML et CSS pour comprendre comment structurer le contenu et le présenter de manière visuellement attrayante. Chaque ligne de code représente un pas de plus vers la maîtrise de ces langages essentiels.

JavaScript :

Plongez dans le monde interactif du JavaScript. Suivez mes aventures alors que j'utilise ce langage pour rendre mes pages web dynamiques et réactives. Des animations aux interactions utilisateur, découvrez comment le JavaScript transforme une page statique en une expérience immersive.

SQL (Structured Query Language) :

Au cours de mon voyage dans le monde du développement web, j'ai également exploré le langage SQL. SQL est un langage de requête utilisé pour interagir avec les bases de données relationnelles. Grâce à SQL, j'ai appris à créer, à modifier et à interroger des bases de données, ainsi qu'à gérer efficacement les données stockées.
Comprendre SQL est essentiel pour tout développeur web, car il permet de manipuler les données de manière efficace, d'effectuer des opérations complexes et d'optimiser les performances des applications web. Grâce à mes expériences avec SQL, j'ai pu approfondir mes compétences en développement web et enrichir mon parcours d'apprentissage.

PHP :

Explorez les coulisses avec PHP, le langage côté serveur. Plongez dans mes scripts pour comprendre comment gérer les données, créer des sessions utilisateur et interagir avec des bases de données. C'est ici que le côté pratique du développement web prend vie.

Explorez et Découvrez :

Explorez, découvrez, et n'hésitez pas à partager vos impressions. Chaque code partagé représente une étape de plus vers la maîtrise du développement web.

Bienvenue dans cet univers. Profitez du voyage !
